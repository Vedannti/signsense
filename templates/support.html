Help Topics

How do I start hand detection?
Go to the Detect page and allow camera access to start real-time detection.
How do I use the Learn page?
Click any alphabet to view hand sign images and zoomed views.
How does the game work?
You must identify the correct alphabet based on the shown gesture.
Which frontend technologies are used?
HTML, CSS, and JavaScript with a responsive UI.
Which backend framework is used?
Flask (Python) is used for routing and server-side logic.
How is feedback stored?
Feedback is stored persistently using SQLite database.
How does gesture recognition work?
Hand landmarks are detected using computer vision and ML models.
